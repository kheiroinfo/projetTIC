Chapitre 4.
Internet et Web

1 ) Introduction

Le web et internet c’est pareil :

Le World Wide Web (WWW) est un système hypertexte public sur Internet . Il a été inventé par Tim Berners-Lee et Robert Cailliau après Internet. Les pages web sont accessibles sur des sites et sont liées entre elles par des hyperliens, d'où l'image de la toile d'araignée. Le Web est l'une des nombreuses applications d'Internet.

  • Internet : support de communication général,
  • Web:une partie des contenus circulant sur l’Internet.

Principe World Wide Web :

  • Accéder à des documents textuels
  • Situés sur des machines accessibles par Internet
  • Reliés entre eux par un mécanisme de lien « hypertexte »
  • Actuellement : servir des ressources
  • De différentes natures : texte, image, son, vidéo, contenu applicatif...et interactif

    • URL (pour Uniform Resource Locator)

    Un lien hypertexte est un lien qui permet de suivre une URL . URL, est le nom unique de la page. décrivant l’emplacement d’une ressource. elle contient :

  • L’adresse du serveur,et l’adresse de la page sur le serveur.

    • 2 )Aspects techniques du Web

    Les 3 mécanismes de base du Web sont :

    1 )URL:

    a. Le Web permet d’accéder à un ensemble de ressources,
    b. Le mécanisme de localisation peut faire appel au protocole DNS

    2 )HTTP:

    a. Protocole de niveau applicatif
    b. Paradigme client-serveur
    c. Protocole sans état

    3 )HTML:

    a. Langage de description de « pages Web »
    b. Texte, images et autres objets
    c. Liens hypermédias entre les pages
    d. Programmation déclarative

    3 )Page web et site web :

    Une page Web, c’est un document multimédia : images, textes, possibilité d’interaction, liens ...etc.
    Décrit élément par élément : titre, morceaux de texte, images...etc.

  • Avec un langage de description :HTML (1992), XHTML (2002) ou HTML5
  • Stocké dans un fichier : html
  • Une page Web est interprétée et affichée par un navigateur, elle est localisée surInternet à l’aide d’une adresse (URL) ;
  • Permet d’accéder à d’autres pages en suivant des liens
  • Un site web est un regroupement de pages Web autour d’une thématique commune, reliées entre elles par des liens hypertextes
  • Emanant d’une même entité , accessibles à partir d’une page d’accueil commune et à partir d’une URL de base commune
  • En général, localisées sur un même serveur
  • Structuré sous forme d’arborescence de dossiers et de fichiers
  • Permet de définir des chemins relatifs

    • 3 )Création d’une page web HTML :

    HTML est :

  • Un fichier texte, construit suivant une syntaxe précise
  • Structuré par des balises
  • Interprété dans un navigateur
  • Créé dans un éditeur de pages web ou tout éditeur de texte

    • Les principes de langage HTML:

    Les balises:

    Un fichier HTML doit comporter au minimum ces 4 balises :

    ...
    <html> ... </html>
    <head> ... </head>
    <title> ... </title>:
    <body> ... </body>

    Les principes de langage HTML:

    <html> ... </html>: Délimite le début et la fin du document
    <head> ... </head>: Entête du document
    <title> ... </title>: Titre du document
    <body> ... </body> : Corps du document

    Les titres et paragraphes:

    <hn> ... </hn>: Titre de niveau n :1 à 6.
    <hn> ... </hn>: Exemple <h1> Grand titre </h1>:
    <p> ... </p>: paragraphe
